Note: Please lower your speaker volume

Application to create Virtual Serial Port
VSPE Download Link: http://www.eterlogic.com/Downloads.html 

Connection: serial1 = directserial realport:com1
	    serial1 = directserial realport:com2

Run program
1. mount c: c:\itcs_code
2. c:
3. main.exe

Main Menu
F1 	- Proceed to the Chat Module
F2 	- Proceed to the Sound Module
F3 	- Proceed to the Game Module
ESC 	- Exit from the system.


Chat Module
Character Accepted 	- ‘   ‘  -  ‘ ~ ’
Emoji 			- ‘/’, ‘*’, ‘-’
Backspace 		- Delete message
Enter 			- Send message to another user


Sound Module
F1 	- Play connected sound
F2 	- Play notification sound
F3 	- Play disconnected sound
F4 	- Access to sound setting to change the sound set
ESC 	- Exit to Main Menu


Game Module
Leftwards Arrow 	- Move left
Rightwards Arrow 	- Move right
Spacebar 		- Shoot arrow
Q 			- Exit to Game’s Menu
